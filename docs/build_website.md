步骤：

### 1.创建网站仓库
#### 1.1 新建仓库
+ 仓库名：按照 `<font style="color:rgb(61, 65, 68);background-color:rgb(250, 250, 250);">你的用户名.github.io</font>`<font style="color:rgb(61, 65, 68);">的格式来写，这里的gorgeousu为用户名</font>
+ <font style="color:rgb(61, 65, 68);">设为公开：Public</font>
+ <font style="color:rgb(61, 65, 68);">添加初始化文件：Add README</font>

![](https://cdn.nlark.com/yuque/0/2026/png/35656643/1769762278111-e8cfdece-234a-4fb0-91f8-3e0f7b7c7e7f.png)

#### 1.2 使仓库生效
在仓库页面设置页面

+ 选择Settings -> Pages
+ 在Build and deployment中选择Branch
+ 点击Save保存

![](https://cdn.nlark.com/yuque/0/2026/png/35656643/1769765352470-94547194-dfe5-4407-8159-8557b8b5f872.png)



### 2.编写网站内容
#### 2.1 方法一：创建简单模板
在仓库页面创建文件

![](https://cdn.nlark.com/yuque/0/2026/png/35656643/1769765089951-002dbf0c-42c3-4d75-9219-bf3e0a421a22.png)

文件名：index.html

文件内容：

```html
<!DOCTYPE html>
<html lang="zh-CN">
  <head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>我的个人网站</title>
  </head>
  <body>
    <h1>你好，世界！</h1>
    <p>欢迎来到我的第一个个人网站。这里是使用 GitHub Pages 搭建的。</p>
  </body>
</html>
```
